Chat UI

Модуль для отображения чат интерфейса

Примеры использования

Модуль может создавать, как окно чата и отображать диалог вместе со строкой ввода для сообщений, так и рендерить только диалог и разработчик сам будет добавлять все фразы в зависимости от бизнес логики его приложения.

  • Полная версия
  • Встроенный чат - используется только компонент диалога
